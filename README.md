The Purpose of this tool is to generate a Mileage Exception Spreadsheet in the Texas Dallas East Mission (TDEM). The purpose is to track approved mileage exceptions per month. This tool enables the creation of that much faster. In order to use this tool, first download a spreadsheet from the Missionary Portal. It is required for the teaching areas to include in this order: Zone, District, Area, Monthly Distance Allowed, Missionary, Car. Under the Car drop down, select "Car". Then download it. You will then upload this to the app.
